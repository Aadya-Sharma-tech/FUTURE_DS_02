## 1. DASHBOARD OVERVIEW
   This dashboard provides a comprehensive view of social media campaign performance across multiple companies, channels, languages, audience groups, and campaign types. It is designed to help marketing teams evaluate engagement, impressions, clicks, ROI, acquisition costs, and audience behavior across different platforms and demographics. It supports strategic decision-making by making performance patterns easily visible.
